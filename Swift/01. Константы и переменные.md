
`import` - импортирует библиотеки для работы с элементами интерфейса например `import UIKit`

`var` - используется для объявления\создания переменных.
`let` - используется для создания констант

```swift
import UIKit

var greetingOne = "Привет"
let greetingTwo = "Привет еще раз!"
print("Привет!")
greetingTwo
```
